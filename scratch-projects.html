Tools Needed: Just go to scratch.mit.edu (free, no download).

Detailed Step-by-Step Instructions

1. Interactive Nepal Tourism Game

  1. Start new project on scratch.mit.edu. Delete cat sprite.
  2. Add 4-5 backdrops: Mountains/Nature + upload free images (Everest, Pokhara lake, Pashupatinath, Lumbini, Chitwan).
  3. Add main "Guide" sprite (tourist character).
  4. Add 4-5 place button sprites (mountain, lake, temple icons) with text costumes.
  5. For each place sprite: When clicked → switch backdrop + ask quiz question (e.g., "Highest mountain?") + if correct change score by 10.
  6. Make variable "score" and show it.
  7. Add start screen: Welcome message + fun sounds on correct answer.
  8. Test and add ending if score > 40.

2. Smart Waste Management Simulator

  1. Delete cat. Add 3 bin sprites (Recycle-green, Organic-brown, E-waste-red) at bottom.
  2. Add 6-8 trash sprites (bottle, peel, phone, paper – upload small PNGs or draw).
  3. For each trash: Forever loop → follow mouse when touching pointer.
  4. Check if touching correct bin → hide + +10 score + cheer sound; wrong → -5 score.
  5. Make "score" variable. Add Nepal message when score high.
  6. Backdrop: Kathmandu river photo + "Help clean Nepal!" text.
  7. Test dragging items.

3. Solar System Explorer

  1. Backdrop: Space image.
  2. Add Sun in center (big yellow circle).
  3. Add 5-8 planet sprites (from library or draw) around Sun.
  4. For each planet: Forever → turn right 1 degree (simple orbit).
  5. When clicked: Say fact (e.g., "Earth: पानी र जीवन छ") + grow bigger for 3 sec.
  6. Add title sprite + space sounds.
  7. Test clicking planets.

4. Digital Storybook on Nepali Culture

  1. Make 5-7 backdrops: Dashain/Tihar images (tika, lights, family – upload free).
  2. Add "Next" button sprite + narrator character.
  3. Variable "page" starts at 1.
  4. When Next clicked: Change page + switch backdrop + narrator says line (record voice or text say).
  5. Add 4-6 pages: Intro Dashain → Tihar → Moral → End.
  6. Add sounds/music on page change.
  7. Test story flow.

Extra Tips: Use Nepali + English text. Add class name & "ICT Fair 2082" on title screen. Practice demo explanation!
